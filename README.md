# 多选文字 (Multi Select)

Chrome 浏览器插件：按住 **Alt** 拖拽可添加选区，不打断之前的选择，支持跨行多选。

## 功能

- **多选累积**：按住 Alt，左键拖拽选中文字，新选区会追加到已有选区，不会替换
- **跨行支持**：可以自由跨行选择
- **复制**：选中后按 Ctrl+C 复制所有选区到剪贴板
- **清除**：点击空白处（不按 Alt）或按 Esc 清除所有选择

## 安装

1. 打开 Chrome，地址栏输入 `chrome://extensions/`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本插件的 `rect-select` 文件夹

## 使用

1. 在任意网页上，**按住 Alt**
2. 左键拖拽选中第一段文字，松开
3. 继续按住 Alt，拖拽选中第二段、第三段… 新选区会追加，不会替换
4. 按 **Ctrl+C** 复制所有选中的文字
5. 点击其他地方（不按 Alt）或按 **Esc** 清除所有选择

## 注意事项

- 滚动页面后高亮会重新计算位置
- 某些受保护的页面（如 Chrome 商店、扩展管理页）可能无法使用

## 技术说明

Chrome 原生选区只能有一个，本插件在 Alt+拖拽时拦截选区，将每次选中的内容存入数组，用 CSS overlay 高亮显示，复制时合并输出。
